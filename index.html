Ресурсы Для Изучения Слово Божия

Есть много ресурсов для изучения Библии на Английском. На Русском так себе. Я хочу это исправить. Есть пару идей для различных ресурсов. Вы сможете их найти тут на этом сайте. На данный момент ничего еше нет и возможно не совсем скоро будет. Работа тут только наченаетса. Работаем сколько можем по возможности.

Текущие Проекты

RST Bible API

API для Русского Синодального перевода Библиию. Сможет принемать книгу, главу, и номер стиха и выдовать текст стиха. Если нада несколька стихов сможет принять вторую главу и стих чтоб выдать текст между первым и вторым стихом.

Будушие Проекты

Стихи По Теме

Списак различных тем и возможность найти стихи которые подходят под тему. Можно голосовать какие стихи лучше подходять. Стихи у которых больше голосов будут с верху. Также можно добавлять стихи которых еше нет.
